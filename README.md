## Featured Projects

### [Hallucination Hunter](https://github.com/Skwork75/Hallucination-Hunter)

AI-powered LLM verification system designed to detect factual hallucinations in generated summaries.

Built a RAG-style verification workflow that compares LLM-generated content against trusted source documents, performs semantic comparison using the OpenAI API, and flags unsupported claims with structured explanations.

**Tech Stack:** `Python` `OpenAI API` `RAG-style Workflow` `Prompt Engineering` `Semantic Comparison`

---

### [Lost & Found Portal](https://github.com/Skwork75/lostfound)

A full-stack community platform for reporting lost and found items with user authentication and item matching logic.

Developed the backend with Django and MySQL, integrated it with a React-based frontend, and designed data models for item categories, location metadata, status tracking, and user contact flow.

**Tech Stack:** `Django` `MySQL` `React` `HTML` `CSS` `JavaScript`

---

### [Expense Tracker](https://github.com/Skwork75/expense-tracker)

A backend application for recording, categorizing, and summarizing personal expenses.

Built using Django ORM with data validation, aggregation queries, and monthly financial summaries.

**Tech Stack:** `Python` `Django` `Django ORM`
